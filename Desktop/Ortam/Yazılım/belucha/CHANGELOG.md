# Changelog

All notable changes to Belucha will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2024-11-04

### Added
- ✅ Product search functionality with SearchBar component
- ✅ Pagination for products in shop page
- ✅ Product reviews display and submission
- ✅ Wishlist functionality with WishlistContext
- ✅ Order detail page with full order information
- ✅ Loading and Error components for better UX
- ✅ Enhanced error handling throughout the application
- ✅ Improved shop page with filters and pagination
- ✅ Search integration in Navbar

### Improved
- ✅ Better error handling and loading states
- ✅ Enhanced product detail page with reviews
- ✅ Improved shop page with search and pagination
- ✅ Better user experience with loading indicators

### Fixed
- ✅ Fixed product fetching queries
- ✅ Improved error messages
- ✅ Better state management

## [1.0.0] - 2024-11-04

### Added
- Initial release of Belucha e-commerce platform
- Multi-vendor support with seller dashboard
- Customer account management
- Admin panel via Payload CMS
- Stripe payment integration
- Shopping cart functionality
- Product catalog with categories
- Order management system
- Review and rating system
- Blog/Pages system
- Multilingual support (EN/DE)
- Responsive design with TailwindCSS
- Framer Motion animations
- SEO optimization
- Image upload support (Media collection)
- Authentication with JWT
- Role-based access control

### Features
- Frontend: Next.js 14 with App Router
- Backend: Node.js + Express + Payload CMS
- Database: MongoDB (via Atlas)
- Payment: Stripe Checkout
- Styling: TailwindCSS
- Animations: Framer Motion
- Forms: React Hook Form + Zod
- i18n: next-intl (EN/DE)
